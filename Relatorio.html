Reconhecimento de Padrões

Problemas do índice de cobertura vegetal de um microhabitat e classificação quanto a sua densidade.

  1. Descrição da base de dados

Para o problema: cobertura vegetal de um microhabitat e classificação quanto a sua densidade, a base contém imagens da copa de vegetações, imagens essas de microhabitat de florestas de densidades diferentes. Para uma maior “justiça”  quanto classificação, ou seja, dados obtidos sob mesmas condições para que seja evitado classificações absurdas. As imagens foram selecionadas pelos seguintes critérios:

  1. Imagens obtidas de cima para baixo com somente vegetação de florestas, campos e bosques.
  2. Ângulo da imagem da posição da inclinação para onde a câmera está apontada entre 60° e 90°.
  3. Imagens diurnas para evitar um tratamento para iluminação seja limiarização global ou variável ou outro tipo de tratamento.
  4. As imagens foram redimensionadas com mesmo tamanho mantendo a proporção.

  1.  Descrição do método

Foi utilizada técnicas de visão computacional para obtenção de informação das imagens, fundamentadas em algoritmos mais utilizados para reconhecimento de padrões em bases de dados.

O método foi dividido em pré-processamento, segmentação/detecção de bordas e classificação.

Na etapa  de  pré-processamento  ocorrem  processos  como  a redução de ruídos  e  o  realce  das  imagens com  o  intuito  de  aumentar  a qualidade  da  imagem  para  que  as  etapas  posteriores não  sofram  as interferências  de possíveis imperfeições.

A segmentação foi usada para isolar regiões da imagem (conjunto de pontos) de interesse por serem pertencentes a objetos ou partes importantes obtidas pela avaliação da descontinuidade dos níveis de cinza para posterior análise, no caso estudado as regiões de interesse foram as vegetações. Foi então usada técnicas de limiarização a saber:  limiarização binária, limiarização adaptativa e limiarização por otsu.

A detecção de borda neste trabalho foi utilizada para identificar objetos no caso específico de vegetação na qual baseados na similaridade dos níveis de cinza podemos classificar a região de interesse na qual quando existir mais objetos teremos mais densidade ou menos densidade  de vegetação e assim podemos fazer uma classificação.

Com as imagens prontas para seu melhor aproveitamento foi então feito testes com os principais algoritmos de classificação de  aprendizagem de máquina obtendo assim os resultados para análise de qual técnica obteve o melhor resultado para resolução do problema.

        Foi utilizado a linguagem Python, juntamente com a biblioteca OpenCV. Algumas bibliotecas python também foram necessárias Numpy (Numeric Python), Scypy (Cientific Python), sklearn (biblioteca de aprendizado de máquina) utilizada para classificação  e métricas de acurácia e Matplotlib biblioteca que foi utilizada para plotagem de gráficos de descrição dos experimentos.

  1. Descrição dos experimentos

Para os experimentos foram feitas variações dos métodos quanto a segmentação/detecção de bordas e classificação. Além de uma análise empírica da base de dados.

Análise empírica:

  1. As imagens obtidas têm gráficos característicos quanto a sua densidade diferindo da intensidade de variações no qual os gráficos menos densos têm picos nas regiões mais claras e mais densos nas regiões mais escuras.
  2. Dependendo do tipo de textura as vegetações com maiores intensidades tendiam a ser mais escuras. Porém o sol interfere nessa análise tendo picos  nas regiões mais claras até em microhabitats mais densos.

          Pré-processamento :

  1. Redimensionamento das imagens: para que elas tenham as mesmas dimensões mantendo a proporção.
  2. Escala de cinza: a imagem foi tratada em escala de cinza para a utilização dos métodos de segmentação e detecção de borda.

Segmentação/detecção de bordas:

         Foi utilizado métodos baseados na descontinuidade dos níveis de cinza (segmentação) e métodos baseados na similaridade dos níveis de cinza (detecção de bordas).

        Na segmentação a ideia foi utilizar a limiarização para que em imagens com microhabitat mais densos e menos densos  tivessem uma diferença suficiente que ajudasse na classificação. Os métodos utilizados foram limiarização, limiarização adaptativa e limiarização Otsu. Já na detecção de borda o importante era identificar a quantidade de objetos na cena onde quanto mais objetos temos mais densidade de vegetação no microhabitat teremos, os métodos utilizados foram Sobel, laplaciano e Canny.

Thresholding (limiarização  binária)

  1. Foram utilizadas imagens pré-processadas em tons de cinza para imagens preto e branco onde todos os pixels possuem 0 ou 255 como valores de intensidade.
  2. Foi usado cv.GaussianBlur para efeito de suavização para retirar ruídos.
  3. Foi testado threshold para limiarização valor de T 50 parâmetro cv2.THRESH_BINARY para binarização e com parâmetro cv2.THRESH_BINARY_INV  para binarização inversa.

Threshold adaptativo

  1. Foram utilizadas imagens pré-processadas em tons de cinza para imagens preto e  branco  onde todos os pixels possuem 0 ou 255 como valores de intensidade.
  2. Foi usado cv.GaussianBlur para efeito de suavização para retirar ruídos.
  3. Foi calculado matematicamente limiar ou seja calculado nos pixels próximos das imagens pela biblioteca. Outro parâmetro é um inteiro que é subtraído da média calculada dentro da caixa para gerar o limiar final.
  4. Foi testado utilizando cv2.ADAPTIVE_THRESH_MEAN_C,  com a imagem suavizada binarizada pela média e a imagem binarizada com Gausse o foi utilizado, cv2.ADAPTIVE_THRESH_GAUSSIAN_C.

Threshold Otsu

  1. Foram utilizadas imagens pré-processadas em tons de cinza para imagens preto e  branco  onde todos os pixels possuem 0 ou 255 como valores de intensidade.
  2. Foi usado cv.GaussianBlur para efeito de suavização para retirar ruídos.
  3. Com método Otsu foi analisado o histograma da imagem para encontrar os dois maiores picos de intensidades, então calculado um valor para separar da melhor forma esses dois picos para isso cv2.THRESH_OTSU.

Sobel

  1. Foram utilizadas imagens pré-processadas em tons de cinza para imagens preto e  branco  onde todos os pixels possuem 0 ou 255 como valores de intensidade.
  2. cv2.Sobel(img, cv2.CV_64F, 1, 0) para transformada em X cv2.Sobel(img, cv2.CV_64F, 0, 1) para transformada em Y.
  3. np.uint8(np.absolute(sobelX)) convertendo para uint8  para Xnp.uint8(np.absolute(sobelY)) convertendo para uint8  para X.

OBS.: Foi juntado o filtro horizontal e o vertical para ter uma transformação completa.

Para suporta valores positivos e negativos Sobel foi preciso trabalhar com a imagem com ponto flutuante de 64 bits para depois converter para uint8 novamente.

laplaciano

  2. Foi utilizada a função cv2.Laplacian(img, cv2.CV_64F) para calculo em 64 bits e  np.uint8(np.absolute(lap)) para conversão para  uint8.

Canny

  1. Foram utilizadas imagens pré-processadas em tons de cinza para imagens preto e  branco  onde todos os pixels possuem 0 ou 255 como valores de intensidade.
  2. Foi usado cv.GaussianBlur para efeito de suavização para retirar ruídos.

  1. Conclusão

        

Os resultados experimentais mostraram que a técnica de segmentação e detecção de bordas tiveram uma diferença significativa nos resultados com maior acurácia para os métodos de detecção de borda. Resultados experimentais mostraram ainda  que o método de sobel obteve a maior acurácia geral .  Foi obtido resultado mais  satisfatórios com os métodos de classificação no qual foi usado métodos baseados na similaridade dos níveis de cinza (detecção de bordas). E resultados em alguns casos  de métodos baseados na descontinuidade dos níveis de cinza (segmentação) foram insatisfatórios o que ocorreu foi que a detecção de objetos(detecção de borda) em cena para as vegetações aguçou as diferenças entre as imagens com vegetação, com suas ricas texturas, mais densa e menos densa enquanto a limiarização acabou deixando imagens com a diferença de densidade apenas baseadas na intensidade dos pixels transformado imagem com base no limiar o que mostrou-se ter menos informações para classificação no caso do probemas específico e na base testada.

Para o thresholding(limiarização) obteve resultados similares tanto para thresholding e o Thresholding invertido para limiares similares.

Thresholding Adaptativo com suavização gaussiana obteve uma ligeira vantagem sobre o Thresholding Adaptativo com suavização mediana quando meto de classificação foi árvore de decisão sendo superior ao método thresholding o que ocorreu é que a imagem tratada teve um ganho significativo de informações o que ajudou na classificação

Tanto Threshold Otsu quanto Thresholding Otsu com Gauss obtiveram uma acurácia muito baixa o que ocorreu foi ao  e analisa o histograma da imagem para encontrar os dois maiores picos de intensidades, então ele calcula um valor para separar da melhor forma esses dois picos, nas imagens seja densa ou não densa tivemos as vezes um pico  luminosidade e esse método na regra de negócio específica não obteve os piores resultados.          

Os métodos de detecção de bordas  Sobel,laplaciano e Canny conseguiram alcançar os melhores resultados na classificação de histogramas gerados nesses casos ocorreu em geral o mesmo fenômeno a saber:A similaridade dos níveis de cinza alcançou mais informações do que apenas verificar as variações abruptas da imagem(descontinuidade dos níveis de cinza) o que ajudou a classificar a imagem como densa ou não densa pois dependendo do limiar árvores muito próximas foram agrupadas como uma só informação enquanto que na detecção de borda essas mesma árvores foram tratada como objetos diferentes tendo assim um ganho de informação.

 Com base nessas informações é mais indicado os métodos de detecção de borda para o problema do índice de cobertura vegetal de um microhabitat e classificação quanto a sua densidade , e para  base específica a indicação é o tratamento da imagem por sobel antes da classificação. A base poderia ter um ganho maior de qualidade com uma quantidade maior de imagens e ainda caso fossem obtidas sob mesmo  hardware. Apesar disso o estudo mostra uma tendências quanto a classificação de imagens para índice de cobertura vegetal com base apenas nas variáveis estudadas

 Com isso pesquisadores podem utilizar a técnica para verificar o índice de cobertura vegetal de um microhabitat e assim usar para classificação quanto a sua densidade.
